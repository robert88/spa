这个dialog控件是基于jquery主要实现以下功能：

一、内容拷贝

内容区有id会console出来

	$.dialog($("#dialogContent"))
#dialogContent

二、内容填充

	$.dialog("<span >这个是一个文件</span>",{title:"<span class="tDanger">this is title!</span>"});

三、下载内容

		$.dialog("url:/load/index.html",{title:"this is title!"});

四、参数设置

  21. 关闭之前之后触发的函数

    this指向参数对象，参数是dialog窗体

    	$.dialog('hello world!',{closeBefore:function(){alert('关闭之前！'+this.find(".dl-body").html()),closeBefore:function(){alert('关闭之后！')+this.find(".dl-body").html()}});

  22. 自定义动画

    自定义动画

    getCenter工具方法获取对象居中坐标

    	$.dialog.fadeLeft=function($dialog){
			var center = $.dialog.getCenter($dialog)
			var w = $dialog.width();
			$dialog.css({left:-w,top:center.y}).stop(true,true).animate({left:center.x},200);
	}
	$(".trigger5").click(function(){
		$.dialog('hello world!',{animateType:'fadeLeft'});
	})

  23. 全局关闭之前之后触发函数

    this指向参数对象，参数是dialog窗体

    这里关闭默认是关闭类型为dialog

    	$.dialog.closebefore = function(){alert('关闭所有之前时触发一次')};
	$.dialog.closeCallBack = function(){alert('关闭所有之后时触发一次')};
	$(".trigger1").click(function(){
		$.dialog('hello world 1!',{mask:false,closeBefore:function($dialog){alert('关闭之前！'+$dialog.find(".dl-body").html())},closeAfter:function($dialog){alert('关闭之后！'+$dialog.find(".dl-body").text())}});
		$.dialog('hello world 2!',{mask:false,closeBefore:function($dialog){alert('关闭之前！'+$dialog.find(".dl-body").html())},closeAfter:function($dialog){alert('关闭之后！'+$dialog.find(".dl-body").text())}});
	})
	$(".trigger2").click(function(){

		$.dialog.closeAll();
	})
	$(".trigger3").click(function(){

		$.dialog.closeFrist();
	})
	$(".trigger4").click(function(){
		$.dialog.closeLast();
	})
